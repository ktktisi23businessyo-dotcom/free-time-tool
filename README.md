# 自由時間 可視化ツール（MVP）

## 概要
このツールは、  
**1日のうち「本当に自由に使える時間」がどれくらいあるのか**を  
数値で可視化するためのシンプルなWebツールです。

睡眠・仕事・通勤・食事など、  
「生きているだけで必ず使われる時間」を差し引いた結果、

> 自己投資や自由に使える時間は、意外と少ない

という事実を実感することを目的としています。

---

## できること
- 週間の生活時間（テンプレート）を設定できる
  - 平日／休日方式
  - 曜日別方式（月〜日）
- 不規則な日だけ「日別上書き」で修正できる
- 以下を自動で計算・表示
  - 今日の自由時間
  - 直近7日間の自由時間合計
- データは端末内に保存（LocalStorage）
  - アカウント登録なし
  - 外部送信なし

---

## 使い方（3ステップ）

### ① 週間テンプレートを設定
まずは、**普段の1週間の基本形**を入力します。

- 睡眠
- 仕事／学業
- 通勤
- 食事
- 入浴・身支度

※ 人によって毎日発生する時間がある場合は  
「追加固定項目（最大3つ）」として登録できます  
（例：育児・介
